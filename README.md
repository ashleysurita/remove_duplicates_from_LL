# remove_duplicates_from_LL

### Problem:
```
Given a sorted linked list, remove all duplicated values from the list. You must return the head of the updated list.
Examples

Input1: 1 → 1 → 2
Output1: 1 → 2
Input2: 1 → 1 → 2 → 3 → 3
Output2: 1 → 2 → 3
Function Signature

func removeDupesLL(head: ListNode) -> ListNode
Tests

Input1: 1 → 1 → 2
Output1: 1 → 2
Input2: 1 → 1 → 2 → 3 → 3
Output2: 1 → 2 → 3
Input3: 1 → 1 → 1 → 1
Output3: 1
Input4: 3
Output4: 3
```
